# sites

-> unzip this folder and place it in your htdocs folder (if using xampp).

-> start your cakephp from terminal

-> make sure your apache and mysql servers are running.

-> go to your web browser and type the following url

	http://localhost/sites/myfirstapp/


you will be asked to register or login. 

once you log in

you will be given an option to create new topic.

on creating a topic users can start posting in it.

edit and delete options have been provided.

in the database cake.sql attached, under the users table two users have already been registerd admin and regular_user. the password for both the accounts is ‘pass’ . Admin is given the privilege to edit and delete a topic or post while a regular_user is allowed to only post the contents.
